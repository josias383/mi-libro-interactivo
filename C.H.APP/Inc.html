Descubramos el Cuerpo

El cuerpo tiene diferentes partes, como la cabeza, el tronco y las extremidades.

La Cabeza

Aquí está la cabeza, donde están los ojos, oídos, boca y nariz.

* Ojos para ver. * Nariz para oler. * Boca para comer y hablar. * Orejas para oír. * Pelo para cubrir la cabeza.

El Tronco

¡El tronco es la parte del cuerpo donde está!

* El corazón que hace pum, pum. * Los pulmones para respirar. * El estómago para digerir la comida.

Extremidad Superior

Extremidades superiores: Son los brazos. Sirven para abrazar, jugar y comer

Extremidades Inferiores

Extremidades inferiores: Son las piernas. Sirven para caminar, correr y saltar.
